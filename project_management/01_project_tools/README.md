# Programming & Data Analysis Tools

These are used to load, process, and analyse the corpus.

* **Python** – main programming language for the analysis pipeline
* **Jupyter Notebook** – interactive coding and experiment documentation
* **Google Colab** – cloud-based notebook for collaborative coding


**Useful Python libraries:**

* **Pandas** – data manipulation
* **NumPy** – numerical computations
* **scikit-learn** – TF-IDF, PCA, classification models
* **NLTK** – natural language preprocessing
* **spaCy** – advanced NLP processing
* **BERTopic** – topic modelling with embeddings
