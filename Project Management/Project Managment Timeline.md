**Project Management Plan**  
**Project:** Comparative Corpus Analysis of Scientific Abstracts in AI/ML/NLP  
**Start Date:** 16 March 2026 **(TBD)**  
**Submission:** 1 May 2026

[Gantt Chart](https://docs.google.com/spreadsheets/d/1T9H45tSD-OmJ1IWh8s-KOzOio8XLI5gVS5o8-YCF1bA/edit?usp=sharing)

The project is organised into six milestones, each representing a major stage of the research pipeline.

1. **Milestone 1 :- Project Definition(16-20 March)**  
   * Objective:  
     	Define the research question and design the text analytics pipeline.  
   * Tasks  
     * Review dataset documentation (HuggingFace corpus)  
     * Decide field scope (AI / ML / NLP)  
     * Define time periods for analysis  
     * Design text analytics pipeline  
     * Set up GitHub repository and project board  
   * Deliverables  
     * Defined research question  
     * Pipeline architecture  
     * Project repository created  
2. **Milestone 2 :- Data Preparation (21-27 March)**  
   * Objective:  
     	Prepare and structure the corpus for analysis.  
   * Tasks  
     * Load dataset  
     * Extract abstracts and metadata  
   * Preprocess text  
     * Lowercase  
     * stopword removal  
     * Lemmatization  
   * Create time period labels  
   * Perform exploratory data analysis  
   * Deliverables  
     * Clean dataset  
     * Dataset statistics and summary plots  
3. **Milestone 3 :- Text Representation**  
   28 March \- 5 April  
   * Objective:  
     	Implement and compare multiple text representation methods.  
   * Methods to Implement  
     * TF-IDF vectors  
     * N-gram / phrase representations  
     * Sentence embeddings  
   * Tasks  
     * Generate representations for each method  
     * Extract keywords or phrases per time period  
     * Store vector representations for later comparison  
   * Deliverables  
     * TF-IDF matrix  
     * Phrase frequency tables  
     * Embedding representations  
4. **Milestone 4 :- Comparative Analysis**  
   6-12 April  
   * Objective:  
     Identify how research focus has changed over time.  
   * Analysis Methods  
     * Keyword comparison across time periods  
     * Topic modelling (LDA or BERTopic)  
     * Classification experiment (predict period from abstract)  
   * Deliverables  
     * Topic distributions  
     * Keyword evolution results  
     * Classification performance results  
5. **Milestone 5 :- Temporal Drift Visualisation**  
   13-18 April  
   * Objective:  
     Visualise the evolution of research topics over time.  
   * Methods  
     * PCA applied to TF-IDF vectors  
     * PCA or UMAP applied to embeddings  
   * Tasks  
     * Project abstracts into low-dimensional space  
     * Colour by publication year or time period  
     * Identify trajectories and clusters  
   * Deliverables  
     * PCA visualisations  
     * Temporal drift analysis  
6. **Milestone 6 :- Evaluation & Reporting**  
   19 April \- 1 May  
   * Objective:  
     Interpret results and document findings.  
   * Evaluation Criteria  
     * Consistency across methods  
     * Interpretability  
     * Alignment with known research trends  
     * Limitations of methods  
   * Tasks  
     * Compare approaches  
     * Write group report  
     * Produce figures and tables  
     * Write individual reflection  
   * Deliverables  
     * Final report  
     * Visualisations  
     * Individual reflection
